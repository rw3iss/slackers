# Slackers Help

Welcome to the slackers in-app help. Type `/help <topic>` to view a
specific topic.

## Topics

- `friends`   — friends list, contact cards, P2P chat
- `themes`    — built-in themes, custom themes, the editor
- `commands`  — slash command syntax and the command list
- `plugins`   — plugin system, games, weather, plugin config
- `setup`     — first-time configuration and OAuth login
- `p2p`       — secure mode, libp2p, NAT, port forwarding
- `secure`    — encryption details (X25519 + ChaCha20-Poly1305)
- `shortcuts` — keyboard shortcuts and rebinding
- `debug`     — debug logging and troubleshooting

## Quick start

- `Tab` / `Shift-Tab`: cycle focus between sidebar, input, and messages
- `Ctrl-S`: open settings
- `Ctrl-H`: open the keyboard shortcut help panel
- `Ctrl-K`: search channels
- `Ctrl-F`: search messages
- `Alt-N`: open the notifications view
- `/`: focus the input bar (or start a slash command)

For the user-facing feature overview, see `README.md` at the project root.
